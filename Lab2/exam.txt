* 对题目的理解
* 现阶段办公室人群深受久坐、高压困扰。传统的健康记录工具缺乏“理解”能力。本项目通过整合“数据记录”与“大模型（SiliconFlow）技术”，旨在打造一个具备高级审美（Frontend Taste）、能听懂用户诉求并给出专业缓解建议的先锋数字健康管家系统。
* 功能清单
* 1. 核心认证与画像：JWT加密鉴权与个人生理基础数据设置。
* 2. 数据枢纽 (Data Entry)：严格物理边界防护的体征录入（心率/血压/睡眠等）。
* 3. 智能面板 (Dashboard)：基于 12 列异步网格 (Asymmetrical Bento) 设计的实时健康大屏与趋势预警。
* 4. Aura AI 引擎：自动提取最近健康档案作为提示词上下文的大语言模型对话接口。
* 非功能需求
* 1. 审美体验 (UI/UX)：坚决摒弃“AI 模板感”，引入 Github Taste 规范，实现黑曜石质感、非对称大字体排版 (Syne) 与胶片空间感，达成业内顶尖视觉。
* 2. 鲁棒性防线：后端拦截所有非生理逻辑的心率/负数异常提交（基于 Pydantic Field）。
* 3. 安全隐私：密码单向加盐加密；无硬编码，大模型 API 强依赖服务端 `.env` 鉴权保护。
* 系统架构与开发技术
* 前端架构：React 18 + Vite，使用 Framer Motion 驱动微动效与 Recharts 绘制图表。
* 后端栈：FastAPI 异步非阻塞架构，响应极速。搭配 SQLAlchemy 持久化至 SQLite。
* AI 集成：集成兼容 OpenAI 标准库的 SiliconFlow Qwen/DeepSeek 模型。
* 开发理念：严格的前后端分离，RESTful API 驱动。
